The Syntax of CSS

x {a: b;}
x is your selector. a is your property. b is your value. Basically put x selects something, a defines what quality is going to be changed, b defines the change.

The Selector Types

Class is used as a general term, '.name' is the example. ID is spesific to one element, '#name' is the example. Element selectors are used for an HTML element, could be p, div, a, anything that is a HTML element. Universal selectors are for EVERY html element, '*' is the symbol. Attribute selectors are for elements with spesific attributes, element[attribute] is the example.

Div and Span

In the most simple of terms. Div is usually for block formatting. Span is for inline/text formatting.

Psuedo-classes

They are selectors/classes that needs a spesific behavior/specification to happen.

Typography: Font Properties

Used to style fonts.

  • Font Family: The 'look', any typeface
  • Font Style: Defines if it's written in a italic, oblique or normal manner.
  • Font Weight: Thickness of the characters, lighter to normal to bolder, or value from 100 to 900 in which 100 is the thinnest.
  • Font Size: The font size, from xx-small to normal to xx-large, or any value.
  • Font Variant: For small caps

Typography: Text Properties

Used to style the text itself.

  • Text Align: Alignment of the text.
  • Text Decoration: Think of underline, line-through, sub, et cetera et cetera.
  • Text Transform: Case of the text, uppercase, lowercase, none, capitalize
  • Line Space: How much space the line takes.

Box Model

This is a layout type in which you organize block-level elements around to form a coherent design. Basically, website lego. In this spesific website, I used CSS flexbox, which automatically styles it for me

CSS Display

Basically, how it behaves in a sense. Static is just normal positioning according to the text-flow. Relative is positioned to the viewport. Absolute is positioned to the nearest positiioned element. Fixes is also positioned to the viewport but stays there. Sticky is static that turns into fixed through a behavior.

Offset

An offset is the "space" or movement generated from the viewport. If a offset is set to 0, then there is no space between the top and the element. Such example is my navbar.

Float

Allows an elements to be taken out of the normal flow, causing all elements to wrap/adapt to it.

Background

This allows for the styling of the background for any elemet (commonly used for the webpage background.) The shorthand notation is background.

Margins

This defines the spacing between elements. Most of this webpage's elements has margin: auto; to center everything.
